Loaded spools

These are the four physical filaments the mixes are solved against. Loading from the printer uses the colors actually assigned to each toolhead — recipes will match what's really on the machine.

Target colors

Blends are estimated with the same color model the slicer's Mix Effect preview uses (verified against it). Printed results depend on real pigments and will drift from any preview — treat ΔE as a ranking, not a promise, and expect dark and fully saturated targets to be flagged: interleaved bright filaments cannot mix into true black. Enter each recipe in the fork's Edit Mix dialog (Ratio tab); the raw definition string is shown for reference.
